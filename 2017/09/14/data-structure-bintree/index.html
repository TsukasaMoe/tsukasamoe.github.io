前序与中序：

  1. 遇到一个节点，压栈，并遍历其左子树
  2. 左子树遍历结束后，栈顶弹出节点
  3. 对弹出节点的右子树进行先（中）序遍历

区别 => 访问时刻：入栈时访问节点是先序，出栈时访问节点为后续

删除节点

按值删除一个节点，用二分查找递归到需要删除的节点，根据删除节点是否有左右子树分三种情况：

  1. 无左右子树，直接将父节点对其引用置为null。
  2. 只有左子树或者右子树，将父节点对其引用置为其左子树或右子树。
  3. 存在左子树和右子树，在右子树中找最小节点代替被删节点，接着对其右子树的替身节点即最小节点进行删除。
